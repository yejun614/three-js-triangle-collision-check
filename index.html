총알과 폴리곤 충돌 여부 판단 알고리즘

  • 동의대학교 응용소프트웨어공학과
  • 응용수학: 3차원 공간에 있는 삼각형을 총으로 쏘았을 때, 교차여부를 확인하는 프로그램
  • 개발자: 정예준 (yejun614@naver.com)
  • 소스코드 Github 링크
  • 마우스: 좌 클릭(선택), 좌 클릭 + 드래그(카메라 회전), 우 클릭 + 드래그(카메라 이동), 휠(카메라 확대, 축소)
  • ESC: 선택 해제, Z: Gizmo 변경
  • 총알 벡터:
  • 충돌한 삼각형의 개수: 0 개
